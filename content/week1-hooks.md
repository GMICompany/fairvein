## Notes

**Music Strategy:**
- Each post goes to TikTok inbox as draft
- Add trending sound from TikTok library before publishing
- Check "Sounds" tab for current trending audio in beauty/lifestyle niche
- 30 seconds max per post

**Image Generation:**
- Use Canva free tier (templates + stock photos)
- Or Pexels/Unsplash (free stock photography)
- Portrait 1024x1536 (TikTok vertical format)
- Run through `scripts/add-text-overlay.js` for text

**Analytics Tracking:**
- Log each post in `analytics/tiktok-posts.json` immediately after publishing
- Check 24h views next morning
- Update 48h and 72h views in following days
- Use diagnostic framework to decide Day 4-7 hooks
