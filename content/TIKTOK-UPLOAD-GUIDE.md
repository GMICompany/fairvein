## How to Upload TikTok Slideshows

### Step 1: Open TikTok App
- Launch TikTok on mobile (iOS/Android)
- Tap **+** button (bottom center) to create

### Step 2: Select Photo Mode
- Tap **"Upload"** (bottom right)
- Select **"Photo"** mode (not video)
- Choose **6 images** for the post (in order: 1.png → 6.png)

### Step 3: Set Slideshow Duration
- Default: 2-3 seconds per slide (12-18 sec total)
- **Recommended:** 3 seconds per slide = 18 sec total (more readable)
- Adjust in editing mode if needed

### Step 4: Add Music (CRITICAL)
- Tap **"Sounds"** at top
- Search for **trending audio in your niche**:
  - Beauty/lifestyle trending sounds
  - "Oh no" sound (if still trending)
  - Check "For You" page for current hot audio
- **Rule:** Music = 2.9x more engagement (Larry's data)
- **Never post silent slideshows** (they get buried)

### Step 5: Add Caption
- Paste caption from guide below (Post 1, 2, or 3)
- Keep hashtags at end (max 5)
- Make sure "fairvein.com" is visible

### Step 6: Privacy Settings
- Set to **"Public"** (not Friends or Private)
- Allow: Comments, Duet, Stitch (for engagement)
- Turn off: Age restriction

### Step 7: Post as DRAFT (Not Publish Yet)
- Tap **"Drafts"** instead of "Post"
- This saves it to your TikTok inbox
- You can preview, edit music, and publish later

### Step 8: Review Before Publishing
- Watch the draft with music
- Check text is readable on all slides
- Verify music matches vibe
- Make sure link in bio is set (fairvein.com)

### Step 9: Publish at Scheduled Time
- Go to Drafts → select post
- Tap "Post" at the scheduled time:
  - Post 1: Day 1, 7:30 AM SGT
  - Post 2: Day 2, 4:30 PM SGT
  - Post 3: Day 3, 9:00 PM SGT

## Pro Tips

### Music Selection
- **Don't use copyrighted music** (audio gets muted)
- **Use TikTok's sound library** (all cleared for use)
- **Trending > Popular** (trending sounds get algorithm boost)
- **Match the vibe:** Shock/drama for Post 1-2, educational for Post 3

### Engagement Strategy
- **Reply to EVERY comment** in first 24 hours (boosts algorithm)
- **Pin top comment** with helpful info or link
- **Engage with similar creators** (comment on beauty/boycott content)

### Analytics Tracking
After posting, track in `analytics/tiktok-posts.json`:
- Post ID, publish date, hook used
- Views at 24h, 48h, 72h
- Likes, comments, shares
- Link clicks (if TikTok analytics shows this)

### What to Watch For
- **High views + Low link clicks** → CTA isn't working (test new slide 6 text)
- **Low views + High engagement** → Hook needs work, content is good
- **High views + High clicks** → SCALE IT (make 3 variations)

## Troubleshooting

**"My post went to drafts but I can't find it"**
- Check TikTok inbox → "Drafts" folder
- Sometimes takes 1-2 min to appear

**"Music sounds off / doesn't sync"**
- Adjust music timing in edit mode
- Make sure music starts at beginning, not 10sec in

**"Text is hard to read on some slides"**
- This is why we use white text + black outline (readable on any background)
- If still unclear, adjust contrast or re-run overlay with darker outline

**"How do I add link in bio?"**
- Edit TikTok profile → Website field → Add: fairvein.com
- Only 1 link allowed (not clickable, users must type it)

**"Post isn't getting views"**
- Wait 24-48 hours (TikTok tests posts gradually)
- Reply to comments to boost engagement
- Check music isn't copyrighted (muted posts get buried)
